# MissedCall AI Landing Page

Static one-page launch funnel for the AI missed-call recovery and booking system offer.

## Open Locally

Open index.html in a browser, or serve this folder with any static server.

## Offer

- Smart lead intake questions
- Booking or quote request page
- Team alerts by email or SMS

- Monthly performance review
